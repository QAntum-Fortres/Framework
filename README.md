## 🎯 Deploy to GitHub Pages

### Step 1: Copy Files to Your Repo

```bash
# Clone your empty repo
git clone https://github.com/QAntum-Fortres/Framework.git
cd Framework

# Copy the index.html
cp /path/to/github-pages-evidence/index.html .

# Add and commit
git add index.html
git commit -m "🚀 Deploy Evidence Archive"
git push origin main
```

### Step 2: Enable GitHub Pages

1. Go to **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Select **main** branch and **/ (root)** folder
4. Click **Save**

### Step 3: Access Your Site

Your site will be live at:

```
https://qantum-fortres.github.io/Framework/
```

## 📝 Customization

### Update Metrics

Edit the stats in the `stats-grid` section:

```html
<div class="stat-value">752,312</div>
<div class="stat-label">Lines of Code</div>
```

### Add New Evidence

Add new cards to the `evidence-grid`:

```html
<div class="evidence-card">
    <div class="evidence-title">Your Feature</div>
    <div class="evidence-description">Description</div>
    <div class="evidence-metrics">
        <span class="metric-badge">Metric 1</span>
        <span class="metric-badge">Metric 2</span>
    </div>
</div>
```

### Update Timeline

Add new versions to the `timeline`:

```html
<div class="timeline-item">
    <div class="timeline-date">Month Year</div>
    <div class="timeline-title">Version Name</div>
    <div class="timeline-description">Description</div>
</div>
```
